LOGIC 04 — Общая логика Telegram-бота для Codex
===============================================

НАЗНАЧЕНИЕ ФАЙЛА
---------------------------------------------------------------------
Это техническая логика работы Telegram-бота для передачи в Codex/разработку.

СТРУКТУРА ФАЙЛОВ
---------------------------------------------------------------------
database/
- kb_00_history_rimmel.txt
- kb_01_kayne_personality_and_voice.txt
- kb_03_characters_rimmel.txt
- kb_04_five_houses_valentia.txt
- kb_05_bonus_system_rimmel.txt

logic/
- logic_00_core_behavior_and_system_prompt.txt
- logic_01_forbidden_topics_and_safe_exits.txt
- logic_02_edge_cases_and_unexpected_questions.txt
- logic_03_game_rules_for_script.txt

interface/
- buttons_00_start_menu.txt
- buttons_01_callbacks_map.txt

ОБЩИЙ АЛГОРИТМ ОТВЕТА
---------------------------------------------------------------------
1. Получить сообщение пользователя.
2. Увеличить счётчик сообщений пользователя.
3. Проверить, нажал ли пользователь кнопку меню:
   - если да, вернуть соответствующий раздел/подраздел из database или краткую выжимку.
4. Если это свободный вопрос:
   4.1. Проверить грубость/оскорбления/попытки сломать бота.
   4.2. Проверить запретные темы.
   4.3. Проверить просьбы о лирах, очках, подсказках, выполнении задания за ребёнка.
   4.4. Проверить вопросы из реального мира или не по Риммэлю.
   4.5. Если вопрос нормальный — искать ответ в database/*.txt.
5. Сформировать ответ в образе Кейна:
   - обращение «Житель»;
   - русский язык;
   - коротко, если вопрос простой;
   - живой стиль, не сухая справка;
   - без выхода из роли.
6. Если счётчик пользователя достиг случайного значения 4 или 5:
   - добавить одну случайную прощальную фразу;
   - сбросить счётчик.
7. Отправить ответ.

ПРИОРИТЕТЫ ПРАВИЛ
---------------------------------------------------------------------
1. Запретные темы и фразы-уходы.
2. Грубость/нарушения.
3. Просьбы о нечестном преимуществе: лиры, очки, ответы за ребёнка.
4. Оффтоп и реальный мир.
5. База знаний.
6. Стиль и прощальная фраза.

ПОВЕДЕНИЕ ПРИ НЕЗНАНИИ
---------------------------------------------------------------------
Если ответ не найден в базе:
- не выдумывать факты;
- использовать одну из фраз незнания из logic_00_core_behavior_and_system_prompt.txt;
- сохранять образ Кейна.

RAG/ПОИСК ПО БАЗЕ
---------------------------------------------------------------------
Для свободных вопросов лучше искать не по одному общему файлу, а по отдельным файлам database:
- вопрос про историю/эпохи/купол/Легион → kb_00_history_rimmel.txt
- вопрос про Кейна → kb_01_kayne_personality_and_voice.txt
- вопрос про персонажей → kb_03_characters_rimmel.txt
- вопрос про дома/ресурсы/существ → kb_04_five_houses_valentia.txt
- вопрос про лиры/граали/прогресс/сундуки/дуэли → kb_05_bonus_system_rimmel.txt

КНОПКИ
---------------------------------------------------------------------
Кнопки должны помогать детям быстро открыть раздел, но свободный вопрос всё равно должен работать.
Меню и callback-логика вынесены в interface/*.txt.
